# Euler_buckling
# Column Buckling (Euler Critical Load)

This script calculates and plots the Euler critical buckling load of a column under different end conditions (fixed-fixed, fixed-pinned, pinned-pinned, and fixed-free). The idea is to see how the effective length factor (K) changes the buckling capacity.

### Features
- Computes Euler critical load:  
  \[ P_{cr} = \frac{\pi^2 E I}{(K L)^2} \]  
- Supports multiple end conditions with different K values.
- Plots \( P_{cr} \) vs. column length for comparison.
- Prints a small table of sample values in the console.
