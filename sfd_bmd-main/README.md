# sfd_bmd
# Shear Force and Bending Moment Diagrams (SFD & BMD)

This script generates shear force and bending moment diagrams for a simply supported beam under two types of loading:
- A single point load at a given position.
- A uniformly distributed load (UDL) across the full span.

### Features
- Calculates support reactions.
- Plots shear force diagram (SFD).
- Plots bending moment diagram (BMD).
- Prints maximum shear and moment values to the console.
