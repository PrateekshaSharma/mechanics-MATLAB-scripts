# Bending_Stress_Distribution
# Bending Stress Distribution in a Rectangular Section

This script computes and plots the bending stress distribution across the depth of a rectangular cross-section subjected to a bending moment.

### Features
- Calculates the second moment of area (I) for a rectangular section.
- Uses the bending equation:  
  \[ \sigma = \frac{M y}{I} \]  
- Plots the stress profile across the depth.
- Highlights extreme fiber stresses (top and bottom).
